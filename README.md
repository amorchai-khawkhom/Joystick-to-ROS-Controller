# Wheelchair Joystick to ROS Robot Controller

Reverse engineered industrial wheelchair joystick system adapted for robotic and ROS-based control applications.

This project includes:
- joystick reverse engineering
- custom PCB interface
- battery monitoring
- embedded signal conversion
- mechanical adaptation
- robotic system integration
